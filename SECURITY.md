# Security Policy

## Supported Versions

Currently supported version for security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please email <aadhar41@gmail.com>.

Please include:

- Description of the vulnerability.
- Steps to reproduce.
- Potential impact.

We ask for discrete disclosure until a fix is released.
